# Dr. Rola Houhou — Academic Portfolio Website

This repository contains the source code for my personal academic portfolio website.

Website: [https://rhouhou.github.io](https://rhouhou.github.io)

The site presents my research and project portfolio in:

- AI for biophotonics
- spectroscopy and coherent Raman imaging
- Raman, CARS, and BCARS signal analysis
- medical imaging AI
- inverse problems
- scientific machine learning
- uncertainty-aware model evaluation
- simulation, reconstruction, validation, and benchmarking workflows

## Main sections

- Home
- Research
- Publications
- Contact

## Project ecosystem

The website highlights my current CARS/BCARS research software ecosystem:

- [CARSBench](https://github.com/rhouhou/CARSBench) — simulation and benchmarking for CARS/BCARS reconstruction under domain shift
- [prCARS](https://github.com/rhouhou/prCARS) — phase retrieval, preprocessing, and Raman-like signal reconstruction
- [CARSGuard](https://github.com/rhouhou/CARSGuard) — validation and quality control for physical realism, Raman consistency, artifact risk, and confidence

## Technical notes

This website is built with GitHub Pages using a lightweight static structure based on Markdown, HTML, CSS, and Jekyll front matter.
